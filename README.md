# Subtitles Manager

## Overview
> **This script just grabs any srt data you create and display using a richtextlabel and bbcode. You can animate by loading a line from the srt using the load_and_display_subtitle_line(index) and then calling the highlight_word_at_index(index) after loading to apply the bbcode setup.**
